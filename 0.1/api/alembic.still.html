add-dependency

fn

[still dependency repositories & {:keys [verbose], :as opts, :or {verbose true}}]

Add a dependency to the classpath. Returns a sequence of maps, each containing a :coords vector, a :jar path and possibly a :current-version string. If the dependency is already on the classpath, returns nil. If the optional parameter :verbose is true (the default), then WARN messages will be printed to the console if a version of a library is requested and the classpath already contains a different version of the same library

distill

fn

[dependency & {:keys [repositories still verbose], :or {still the-still, verbose true}}]

Add a dependency to the classpath.

:repositories : specify a map of leiningen style repository definitions to be used when resolving. Defaults to the repositories specified in the current lein project.

:still : specifies an alembic still to use. This would be considered advanced usage (see the tests for an example).

:verbose : specifies whether WARN messages should be printed to the console if a version of library is requests and there is already a different version of the same library in the classpath. Defaults to true
